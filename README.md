# Автоматизированное обновление цен и остатков товаров на Ozon

Данный скрипт предназначен для автоматизации работы с товарами в интернет-магазине на платформе Ozon. Он позволяет загружать остатки и обновлять цены на товары, используя API Ozon. Скрипт извлекает данные о товарах, обрабатывает их и отправляет обновления обратно в систему Ozon.

Скрипт загружает файл с остатками товаров в формате ZIP, извлекает его и преобразует данные в удобный для обработки формат. Используя API Ozon, скрипт получает список всех товаров с их уникальными идентификаторами, что позволяет сопоставить данные о остатках с товарами в магазине. Скрипт обрабатывает данные о остатках, фильтруя только те товары, которые имеются в магазине, и формирует списки для обновления цен и остатков. Обновленные цены и остатки отправляются обратно в систему Ozon через соответствующие API-запросы.

Скрипт упрощает процесс управления товарами в Ozon, позволяя автоматически обновлять остатки и цены.

# Автоматизированное обновление цен и остатков товаров на Yandex.Market (Partner API)

Модуль автоматизирует обновление остатков и цен на Yandex.Market через Partner API. Он загружает локальный файл с актуальными остатками и ценами, сопоставляет записи с товарами кампании в Маркете и формирует пакет запросов для массовой отправки обновлений.
